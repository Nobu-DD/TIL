### [paizaスキルチェック「C130:繰り返し学習」](https://paiza.jp/challenges/share/5SAXRsarGG4Jrf6QqH5M-xcv-CAz8nthFIbZd-tqb7s?campaign=165&source=social)
#### 自身の解答
```ruby
n = gets.to_i

anothre_n = 0
another_q = []

n.times do |n|
    question = gets.split
    unless question[0] == "y" && question[1] == "y"
        anothre_n += 1
        another_q << n + 1
    end
end

if anothre_n >= 1
    puts anothre_n
    puts another_q
else
    puts anothre_n
end

```
