### git reflog
gitの操作履歴を確認できる。
### git reset --hard HEAD@{n}
reflogでチェックした作業番号を指定すると、該当の作業場所まで戻すことができる。（--hardはワーキングツリーまで全ての変更点を元に戻す）
### git stash push -m "コメント" -- ファイル名 ファイル名1
コメントを打つ場合は-mオプションを使用する。全てのファイルだけでなく個別のファイルを退避したい場合はスペース区切りでファイル名を指定する。
### git reset --soft HEAD^
一つ前のコミットをリセットする。HEAD末尾に^(キャレット)を一つつける必要がある。（HEADは最新のコミット情報）
softにすることで、作業内容を残したままコミットを取り消す。
### git restore ファイル名
指定したファイルのワーキングツリーを最新コミット時の状態に戻す。
