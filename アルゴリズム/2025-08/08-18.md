### [paizaスキルチェック「C013:嫌いな数字」](https://paiza.jp/challenges/share/CNsPA4whbtp0mWXHdtZsmO3jIZx_s9NNbYp48vqn5iE?campaign=165&source=social)
#### 自身の解答
```ruby
dislike = gets.chomp
n = gets.to_i
admittance = []

n.times do
    room = gets.chomp
    unless room.include?(dislike)
        admittance << room.to_i
    end
end

if admittance.count >= 1
    puts admittance
else
    puts "none"
end
```
